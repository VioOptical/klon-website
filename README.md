# Klon Website

Static clone of two VIO Optical landing pages.

## Pages

- `/solusi-mata-minus-vio-google-jbk/`
- `/terapi-mata-minus-ortho-k-jabodetabek-ard/`

## Local Preview

Run a static server from the project root:

```bash
python -m http.server 4175
```

Then open:

```text
http://localhost:4175/
```

## Netlify

This project is configured as a static site. Netlify should publish the repository root with no build command.
